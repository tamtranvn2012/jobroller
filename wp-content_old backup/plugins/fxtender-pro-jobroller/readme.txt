Unofficial plugin for the premium JobRoller theme that extends it's functionality adding new features, options and widgets.

== Description ==

Extends <a href=\"http://www.appthemes.com/themes/jobroller\">JobRoller</a> premium theme with new features, options and widgets. Developed for the power users who want unique features and options for their favourite job portal theme.

After installing *FXtender* on your *JobRoller* theme a new set of features and widgets will be instantly available on a dedicated Admin panel. The majority of these features offers a new set of options that will let you tweak your Job Portal to fit your needs.

**Why FXtender?**

*   New features, options and Widgets
*   Every feature is plugged in. No changes necessary to JobRoller core files. Just install and use.
*   No other changes needed
*   No child theme needed

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page. 
2. A new Sub-Menu named 'FXtender' will be available under the JobRoller menu.
3. Just activate the features you want under the new created sub-menu panel.
